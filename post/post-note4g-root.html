Mi Note 4g v6.2.2 မှာ root နည်း !!!

Mi Note 4g v6.2.2 တမှာ အလွန်လွယ်ကူသောနည်းနဲ့ rootလို့ရနေပါပြီ။ နည်းလမ်းကတော့

၁. Root ဖိုင်ကို ဒေါင်းပြီးတော့ internal storage ထဲဖြစ်ဖြစ် sd card ထဲဖြစ်ဖြစ်ထည့်ထားလိုက်ပါ။

၂. Updater App ထဲကို သွားလိုက်ပါ။ ပြီးတော့ ညာဘက်အပေါ်ဆုံးမှာရှိတဲ့ menu ကို နှိပ်လိုက်ပါ။

၃. Select Update Package ကနေ အပေါ်က zip file ကို လမ်ကြောင်းရှာပြီးရွေးလိုက်ပါ။

၄. install လုပ်ပြီးရင် အလိုလို reboot ကျသွားပြီးနောက် ပြန်တက်လာပါမယ်။

၅. Permission App ထဲမှာ Root Access ဆိုတဲ့ Menu လေးပေါ်လာရပါမယ်။
